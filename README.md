# -0100-WEB-page-Blynk
Веб страничка на ESP8266. Можно настроить, подключение к WiFi. Задать данные для Blynk.  Страничка запаролена. Если нет WiFi то ESP8266 поднимет точку доступа AP. И наша страничка будет доступна по адресу 192.168.4.1 Когда доступен роутер то в окружении сети появляется наша ESP это SSDP то есть устройство(адрес web странички) легко найти да же если нет Arduino IDE и Сериал монитор недоступен. Еще поднимается FTP на ESP туда то я и заливаю WEB странички нашего интерфейса. Мало ли для чего еще можно использовать. )))
